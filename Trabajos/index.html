Mermaid es una biblioteca de JavaScript que permite a los programadores y otros profesionales generar diagramas y gráficos a partir de texto en un formato similar al Markdown. Esto puede ser útil para visualizar datos, estructuras, relaciones y flujos de proceso de una manera que sea fácil de entender y compartir. Mermaid es ampliamente utilizado para crear diagramas en documentación técnica, notas, y otras comunicaciones donde la visualización gráfica puede ayudar a explicar conceptos complejos o mostrar relaciones entre componentes. Durante el código se usa simbolo %% para comentarios. El código se pone en portal de Memaid
