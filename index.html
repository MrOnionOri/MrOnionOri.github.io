Componente PN532

El PN532 es un controlador de comunicación NFC (Near Field Communication) altamente integrado para dispositivos RFID y NFC.

Características Principales

  • Compatible con la norma NFC-A, NFC-B, y NFC-F
  • Soporte para lectura y escritura NFC
  • Interfaz de comunicación SPI, I2C, UART y USB
  • Funciona a 13.56 MHz
  • Capacidad para leer etiquetas NFC, tarjetas inteligentes y dispositivos móviles

Aplicaciones

El PN532 se utiliza en una amplia variedad de aplicaciones, incluyendo:

  • Sistemas de acceso sin contacto
  • Pagos móviles
  • Control de acceso
  • Transporte público
  • Automatización del hogar
